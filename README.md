# 🤖 AI Resume Screening System

This project is an AI-powered Resume Screening System built using LangChain.
It evaluates candidates based on a given job description and provides a score along with an explanation.

---

## 🚀 Features

* Extracts skills, experience, and tools from resumes
* Matches resume data with job requirements
* Assigns a suitability score (0–100)
* Generates explanation for the score
* Uses LangSmith tracing for monitoring and debugging

---

## 🧠 Pipeline

Resume → Extraction → Matching → Scoring → Explanation

---

## 🛠 Tech Stack

* Python
* LangChain (LCEL)
* Transformers
* LangSmith

